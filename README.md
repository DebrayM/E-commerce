# E-commerce

## Création template

À partir du wireframe situé sur l'URL suivante (https://whimsical.com/e-commerce-64AcmxQRMUHNF7UvSC4aP3), créer avec Photoshop un template de e-commerce complet.

**Le style, les couleurs, les formes et les produits à vendre sont libres.**

Vous devez impérativement respecter la position des éléments fournis par le wireframe.


## Page "Détails d'un produit"

En gardant l'esprit de votre template, créer une page de détails d'un produit contenant un champ permettant le choix d'une quantité et et un bouton "Ajouter au panier".


## Page "Panier"

Toujours en restant dans le thème du template, réaliser la page "Panier" de votre e-commerce contenant au minimum les informations suivantes :

* Titre de l'article
* Prix unitaire
* Quantité
* Total de l'article
* Total du panier


## Intégrations 

En fois les templates créent, intégrer-les en respectant les normes W3C (code HTML valide)
